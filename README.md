# hlash99.github.io

Personal landing page / dashboard at **https://hlash99.github.io** that links to my projects.

The card grid is generated client-side from the GitHub API, so any repo I add appears
automatically — repos with GitHub Pages enabled show up under **Live sites**, the rest
under **Other repositories**. Per-repo icons and blurbs can be customized in the `META`
map inside `index.html`.
